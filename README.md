# VueJS-with-Vite

This template should help get you started developing with Vue 3 in Vite.

---

To coś ma pokazać, że mam opanowany javascript + html5 na poziomie podstawy podstaw. I przy okazji Vue 3, nawet jeśli używam go jak czystego JavaScriptu.

Od strony UX/UI wygląda okropnie i jest niefunkcjonalne. Istnieją błędy, które nigdy się nie ujawnią jak naciśnięcie przycisku download może pobrać plik ale nie musi. Instrukcje nieczytelne. Nie mniej można by sobie wyobrazić, że oferuje to minimum niezbędnych funkcji.

1. Wpisz adres strony np. <code>example.com</code> w hosts.
2. Jeśli strona znajduje się w folderze np. <code>example.com/folder</code> wpisz /folder w path
3. Z listy po lewej wybierz wszystko, co opisuje treść na Twojej stronie.
4. W niektórych przypadkach musisz podać kontekst. Na przykład nagość może mieć charakter medyczny albo edukacyjny.
5. Jeśli używasz kilku stron z różnymi treściami, na przykład posiadasz osobną kategorię dla dorosłych dodaj nową kartkę (new page).
6. Poszczególne kartki możesz przeglądać i uaktualniać (update).
7. Możesz usunąć ostatnią kartkę (delete).
8. Zalecamy przejrzenie pliku przed pobraniem (preview).
9. Pobierasz przyciskiem download.
10. Więcej informacji jak dodać plik: https://www.w3.org/TR/powder-primer/#ICRA1
